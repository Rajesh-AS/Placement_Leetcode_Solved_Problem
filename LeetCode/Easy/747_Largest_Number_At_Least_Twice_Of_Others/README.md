# LeetCode 747 - Largest Number At Least Twice of Others

**Difficulty:** Easy

## Problem Link

[https://leetcode.com/problems/largest-number-at-least-twice-of-others/](https://leetcode.com/problems/largest-number-at-least-twice-of-others/)

## Approach

Find the maximum element and the second maximum element in a single pass. If the maximum is at least twice the second maximum, return its index. Otherwise, return -1.

## Time Complexity

O(n) — single pass through the array.

## Space Complexity

O(1) — only two tracking variables.
